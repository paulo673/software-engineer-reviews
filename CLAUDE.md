# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a study and review repository for software engineering interview preparation. Content is organized around specific job descriptions, with review materials, notes, and resources grouped by topic.

## Structure Convention

- Each job description lives in its own directory (e.g., `jobs/company-role/`)
- Inside each job directory: the raw JD, plus topic subdirectories for study notes
- Notes are Markdown files; no build system or test runner is needed

## Working in This Repo

- All content is Markdown — prefer clear headings, concise bullet points, and code blocks for examples
- When adding review material for a new JD, identify the distinct skill areas first, then create one file per topic
- Commit messages follow Conventional Commits in English (e.g., `docs: add review notes for distributed systems`)
- Keep the README up to date if the directory structure or workflow changes significantly
