# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a study and review repository for software engineering interview preparation. Content is organized around specific job descriptions, with review materials, notes, and resources grouped by topic.

## Structure Convention

- Content is organized by topic, not by job description
- Each topic gets its own Markdown file (e.g., `distributed-systems.md`, `system-design.md`)
- Job descriptions are used as input to identify which topics to cover, but are not stored as artifacts

## Working in This Repo

- All content is Markdown — prefer clear headings, concise bullet points, and code blocks for examples
- When a JD is provided, extract the distinct skill areas and create or update one file per topic
- Commit messages follow Conventional Commits in English (e.g., `docs: add review notes for distributed systems`)
- Keep the README up to date if the workflow changes significantly
