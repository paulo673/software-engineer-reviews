# Software Engineer Interview Reviews

A personal study repository for preparing for software engineering job interviews.

## Purpose

This repo organizes review materials based on real job descriptions. Each job description drives a focused study plan covering the technical areas, skills, and tools most relevant to the target role.

## Structure

Each job description gets its own directory with:
- The original job description
- Study notes and summaries per topic
- Key concepts, patterns, and references

## How It Works

1. A job description is added to the repo
2. Required skills and technologies are identified
3. Review materials are organized by topic area
4. Preparation notes are built incrementally over time
