# Software Engineer Interview Reviews

A personal study repository for software engineering interview preparation.

## Purpose

This repo is a collection of topics and review materials for software engineering roles. Job descriptions are used as reference to identify what to study, but the content itself is organized by topic — not by job.

## How It Works

1. A job description is analyzed to extract the relevant skill areas
2. Each skill area becomes a topic in this repo
3. Review notes, key concepts, and references are built up per topic over time
